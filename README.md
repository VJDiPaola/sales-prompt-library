# Sales Prompt Library

Reusable AI prompts to accelerate sales workflows. Copy, paste into Claude/ChatGPT, customize, and use.

## How to Use This Library

1. Browse prompts by category below
2. Click on any prompt file to open it
3. Copy the prompt text
4. Paste into Claude or ChatGPT
5. Replace [bracketed variables] with your specifics
6. Save winning variations back to this repo
